# AI Team Plugin

AI 专业团队系统，通过对话式智能调度协调多个专业 subagent 协作开发。

## 快速开始

```bash
# 安装
ln -s /root/dev/set_claude/ai-team-plugin /root/.claude/plugins/ai-team

# 使用
/ai-team 开发用户权限管理
```

## 核心特性

- ✅ 对话式智能调度
- ✅ MCP 工具细粒度控制
- ✅ Beads 任务跟踪集成
- ✅ 三层防护机制
- ✅ Git hooks 自动化
